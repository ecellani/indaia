Alta Performance
Impact
As baterias Impact foram desenvolvidas para sistemas de som automotivo profissional. Diferentes das baterias comuns, elas fornecem energia mais rápido (altos picos), o que é fundamental para sistemas de áudio alta fidelidade e alta potência. Outra característica fundamental para essa aplicação é a reserva de capacidade. Quanto maior ela for, maior será o tempo para você curtir o seu som.
